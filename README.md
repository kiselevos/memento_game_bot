# Telegram Fun Photo Battle Bot

Маленький телеграм-бот на Go, в котором игроки в группе получают задания и присылают фото на тему, после чего голосуют за лучшее фото

## Технологии
- Go
- Библиотека [telebot](https://github.com/tucnak/telebot)
- Telegram Bot API

## Запуск
```bash
export TELEGRAM_TOKEN=your_token
go run main.go
